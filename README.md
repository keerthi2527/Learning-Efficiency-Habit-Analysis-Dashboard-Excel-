# Learning-Efficiency-Habit-Analysis-Dashboard-Excel-
Interactive Excel dashboard analyzing productivity, focus, and daily habit patterns using KPIs and data visualization.

Overview

This project presents a data-driven Excel dashboard that evaluates the impact of daily habits on productivity, focus, and learning efficiency. The solution transforms structured habit data into actionable insights through interactive visualizations and performance metrics.

Business Objective

To analyze how behavioral factors such as study time, sleep duration, mobile usage, and lifestyle activities influence overall productivity and goal achievement, enabling more informed time-management decisions.

Technical Stack

Microsoft Excel

Pivot Tables & Pivot Charts

Slicers for dynamic filtering

Excel formulas for KPI calculations

Dashboard Capabilities

Dynamic Key Performance Indicators (KPIs) for productivity percentage, goal achievement rate, and average study hours

Interactive month-wise and day-wise filtering

Comparative analysis of study, sleep, and focus levels

Evaluation of mobile usage impact on learning efficiency

Planned vs actual performance tracking to assess consistency

Time allocation visualization to identify optimization opportunities

Analytical Insights

Productivity declines as unstructured mobile usage increases

Adequate sleep directly improves focus and study effectiveness

Consistent execution minimizes gaps between planned and actual performance

Minor habit adjustments can yield measurable productivity improvements

Use Cases

Academic performance monitoring

Individual and team productivity analysis

Habit-based performance optimization

Entry-level analytics and reporting demonstrations

Conclusion

This project demonstrates the effective use of Microsoft Excel as an analytical and visualization platform for performance analysis. It highlights how structured data and visual storytelling can support smarter decision-making and productivity improvement.
